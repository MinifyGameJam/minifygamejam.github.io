What is Minify Game Jam?

Minify Game Jam is a solo, hacker-friendly, game development competition held usually once a month. It is a small jam for people with large schedules that can write tiny amounts of code that do amazing things.

Rules

The rules are simple. Create a game, in 24 hours, that runs on Linux and/or PC (sorry mac), that fully implements that months theme. The games will be judged on how much code it took to implement, primarily, and uniqueness adds bonus points. The winner will be on the main page for a month, and get to gloat to all his friends on IRC.
