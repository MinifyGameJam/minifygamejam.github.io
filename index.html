What is Minify Game Jam?

Minify Game Jam is a solo, hacker-friendly, game development competition held usually once a month. It is a small jam for people with large schedules that can write tiny amounts of code that do amazing things.

Rules

The rules are simple. Create a game, in 24 hours, that runs on Linux and/or PC (sorry mac), that fully implements that month's theme. The games will be judged on the following criteria:
  1. how much code it took to implement*
  2. originality
  3. how well it executes its concept
* 3rd party libraries won't be included in amount of code it took to write.(Within reason) if you open source your framework, we won't include that code in our estimation of project minimalism.

Rewards

The winner will be on the main page for a month and get to gloat to all of his or her friends on IRC. Also, the winner gets to experience the pride of being better than everyone else.
